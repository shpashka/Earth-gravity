Физическая симуляция притяжения планеты Земля

Используется физическая формула силы притяжения к Земле в векторном виде. Земля считается сферической, поэтому в силу сферической симметрии не возникает необходимости в интегрировании по элементарным массам. Точка притяжения - центр Земли.

Есть возможность задания вектора начальной скорости(во внутренних координатах pygame, то есть с инвертированной осью у) и координат спутника(см. строку sat = ...)
Также при этой инициализации можно включить параметры do_graph и do_lessen

do_graph=True - после симуляции отображается график отслеживаемого параметра(по умолчанию орбита в км)
do_lessen=True - скорость постепенно немного уменьшается

Есть обработка нажатий на клавиши:
W-сместить спутник вверх
S-сместить спутник вниз
A-сместить спутник влево
D-сместить спутник вправо

вывести в консоль:
1 - расстояние от Земли
2 - модуль скорости
3 - модуль ускорения
4 - вектор скорости в ПДСК
5 - вектор ускорения в ПДСК
